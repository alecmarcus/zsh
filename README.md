# zsh

Dotfiles for zsh configuration. Simple (no frameworks) and portable (plugins as submodules).

> Reference: https://thevaluable.dev/zsh-install-configure-mouseless/

1. Install zsh (no need for macOS) `sudo apt install zsh`
1. [Install fzf](https://github.com/junegunn/fzf#installation) `brew install fzf`
1. `source $ZDOTDIR/.zshrc`
